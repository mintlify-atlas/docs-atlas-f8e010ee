## Overview

StudyQuest's quiz system transforms learning into an engaging game-like experience. Students answer multiple-choice questions with **3 lives**, receive immediate feedback with sound effects, and see their progress through an animated avatar companion.

<Card title="Quiz Features" icon="sparkles">
  - **Lives System**: 3 hearts - lose all and the level fails
  - **Audio Feedback**: Success and failure sound effects
  - **Reactive Avatar**: Visual companion that celebrates or sympathizes
  - **Detailed Explanations**: Learn from mistakes with AI-generated explanations
  - **XP Rewards**: Earn +20 XP for completing quizzes
</Card>

## Answer Checking Logic

### Validation Flow

<Steps>
  <Step title="User Selects Option">
    Taps one of the four multiple-choice options
  </Step>
  
  <Step title="Answer Validation">
    System compares selected index with correct answer index
  </Step>
  
  <Step title="Audio Feedback">
    Plays success.mp3 or wrong.mp3 based on correctness
  </Step>
  
  <Step title="Avatar Reaction">
    Shows animated avatar celebrating or sympathizing
  </Step>
  
  <Step title="Lives Deduction">
    If wrong, remove one heart. If no hearts left, trigger game over
  </Step>
  
  <Step title="Explanation Display">
    Show AI-generated explanation regardless of correctness
  </Step>
</Steps>

### Implementation

```dart quiz_page.dart
void _checkAnswer(int selectedIndex) {
  if (_isAnswered) return;

  final currentQ = _questions[_currentIndex];
  final isCorrectNow = (selectedIndex == currentQ.correctIndex);

  setState(() {
    _selectedOptionIndex = selectedIndex;
    _isAnswered = true;
    _isCorrect = isCorrectNow;
    
    _currentReaction = isCorrectNow 
        ? AvatarReaction.success 
        : AvatarReaction.fail;
    _showAvatarPopUp = true;
  });

  // Play sound effect
  _playSound(isCorrectNow);

  // Lives logic
  if (!isCorrectNow) {
    setState(() {
      _lives--;
    });
    
    if (_lives <= 0) {
      _showGameOverDialog();
      return;  // Stop here - user lost
    }
  }

  // Hide avatar after 1.5 seconds
  Future.delayed(const Duration(milliseconds: 1500), () {
    if (mounted && _lives > 0) {
      setState(() {
        _showAvatarPopUp = false;
        _currentReaction = AvatarReaction.idle;
      });
    }
  });
}
```

## Audio System

Quizzes use the `audioplayers` package for sound effects:

### Audio Player Setup

```dart quiz_page.dart
import 'package:audioplayers/audioplayers.dart';

class _QuizPageState extends State<QuizPage> {
  final AudioPlayer _audioPlayer = AudioPlayer();
  
  // Play sound based on correctness
  Future<void> _playSound(bool isCorrect) async {
    try {
      if (isCorrect) {
        await _audioPlayer.play(AssetSource('sounds/correct.mp3'));
      } else {
        await _audioPlayer.play(AssetSource('sounds/wrong.mp3'));
      }
    } catch (e) {
      print("Error reproduciendo audio: $e");
    }
  }
  
  @override
  void dispose() {
    _flutterTts.stop();
    _audioPlayer.dispose();  // Clean up audio from memory
    _confettiController.dispose();
    super.dispose();
  }
}
```

<Note>
Audio files should be placed in `assets/sounds/` and declared in `pubspec.yaml`
</Note>

## Option Display

Options change color based on correctness:

```dart quiz_page.dart
Widget _buildOptionCard(int index, String text, int correctIndex) {
  Color backgroundColor = Colors.white;
  Color borderColor = Colors.grey[300]!;
  
  if (_isAnswered) {
    if (index == correctIndex) {
      backgroundColor = Colors.green[100]!;
      borderColor = Colors.green;
    } else if (index == _selectedOptionIndex && index != correctIndex) {
      backgroundColor = Colors.red[100]!;
      borderColor = Colors.red;
    }
  }

  return Padding(
    padding: const EdgeInsets.only(bottom: 12.0),
    child: InkWell(
      onTap: _isAnswered ? null : () => _checkAnswer(index),
      borderRadius: BorderRadius.circular(12),
      child: Container(
        padding: const EdgeInsets.symmetric(horizontal: 16, vertical: 16),
        decoration: BoxDecoration(
          color: backgroundColor,
          border: Border.all(color: borderColor, width: 2),
          borderRadius: BorderRadius.circular(12),
        ),
        child: Row(
          children: [
            CircleAvatar(
              radius: 12,
              backgroundColor: borderColor,
              child: Text(
                String.fromCharCode(65 + index),  // A, B, C, D
                style: TextStyle(
                  fontSize: 12, 
                  color: _isAnswered ? Colors.white : Colors.grey[600]
                ),
              ),
            ),
            const SizedBox(width: 12),
            Expanded(
              child: Text(text, style: const TextStyle(fontSize: 16)),
            ),
          ],
        ),
      ),
    ),
  );
}
```

## Quiz Data Entity

```dart quiz_entity.dart
class QuizEntity {
  final String id;
  final String question;
  final List<String> options;
  final int correctIndex;
  final String explanation;

  QuizEntity({
    required this.id,
    required this.question,
    required this.options,
    required this.correctIndex,
    required this.explanation,
  });

  factory QuizEntity.fromMap(Map<String, dynamic> map) {
    return QuizEntity(
      id: map['id'] as String,
      question: map['question_text'] ?? 'Sin pregunta',
      options: List<String>.from(map['options'] ?? []),
      correctIndex: map['correct_answer_index'] as int,
      explanation: map['explanation'] ?? '',
    );
  }
}
```

## Best Practices

<CardGroup cols={2}>
  <Card title="Lives Management" icon="heart">
    Always check `_lives > 0` before allowing progression to prevent edge cases
  </Card>
  
  <Card title="Audio Cleanup" icon="broom">
    Dispose audio players and TTS in `dispose()` to prevent memory leaks
  </Card>
  
  <Card title="Mounted Check" icon="shield">
    Always verify `mounted` before setState after async operations
  </Card>
  
  <Card title="Disable Tap After Answer" icon="hand">
    Prevent multiple answers by checking `_isAnswered` in tap handler
  </Card>
</CardGroup>
